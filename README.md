# Robot Grasp Detection

## Installation

1. Create a new conda environment:
    
    ```bash
    conda create -n gcnn python=3.6
    ```
    
2. And activate the environment:
    
    ```bash
    conda activate gcnn
    ```
    
3. Install the necessary Python packages:
    
    ```bash
    pip install -r requirements.txt
    ```
